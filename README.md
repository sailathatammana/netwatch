## 2. Setup

These are the instructions to run the project:

1. Clone the project :
   `https://github.com/sailathatammana/netwatch`
2. Install dependencies:
   `npm install`
3. run the project:
   `npm start`
4. Open a browser at : `http://localhost:3000`
